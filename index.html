关于hexo-theme-next主题配置”

发表于 2019-11-08 |

此篇内容关于此网页的完善过程，会持续更新。

默认已经搭建好了个人博客

1、在blog目录中打开命令行（鼠标右键Git Bash Here或win+r输入cmd然后让命令行的位置处于blog目录 ）输入 git clone https://github.com/iissnan/hexo-theme-next themes/next 这是将next主题下载到F:\github\Blog\themes。

2、打开Blog根目录将主题修改为next

#二、添加评论功能

之前用过 gitment ,个人感觉不是很友好，后来使用了 Valine 这个评论服务 。

1、注册LeanClound,获取APP ID 和 APP Key（https://leancloud.cn/）
2、注册完以后需要创建一个应用（应用随便取名，我的是mblog），然后 进入应用 -> 设置 -> 应用 key，得到 appid、appkey。
3、在设置页点击存储，查看是否有Comment和 Counter，没有则创建这两个，权限设为无限制。
4、设置页—存储—安全中心将数据储存外其他选项全部关闭，然后将鼠标滑下来，找到Web安全域名，将自己的域名加上去，比如我的是http://zhuangjiel.top/。
5、由于next内置了Valine，这里只需要进行配置# Valine

# You can get your appid and appkey from https://leancloud.cn

# More info available at https://valine.js.org

valine: enable: true # When enable is set to be true, leancloud_visitors is recommended to be closed for the re-initialization problem within different leancloud adk version. appid: # your leancloud application appid appkey: # your leancloud application appkey notify: false # mail notifier, See: https://github.com/xCss/Valine/wiki verify: false # Verification code placeholder: 'ヾﾉ≧∀≦)o来啊，快活啊!' # comment box placeholder avatar: mm # gravatar style guest_info: nick,mail,link # custom comment header pageSize: 10 # pagination size language: zh-cn # language, available values: en, zh-cn visitor: true # leancloud-counter-security is not supported for now. When visitor is set to be true, appid and appkey are recommended to be the same as leancloud_visitors' for counter compatibility. Article reading statistic https://valine.js.org/visitor.html comment_count: true # if false, comment count will only be displayed in post page, not in home page
